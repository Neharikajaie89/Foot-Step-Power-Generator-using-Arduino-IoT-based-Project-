# Foot-Step-Power-Generator-using-Arduino-IoT-based-Project-

A smart **IoT-based Footstep Power Generation System** that converts mechanical energy from footsteps into electrical energy using piezoelectric sensors. The generated power is stored in a rechargeable battery and monitored using an Arduino-based IoT system, allowing real-time voltage and power monitoring.

---

## 📌 Overview

As the demand for renewable energy grows, harvesting energy from human movement has become an innovative solution. This project demonstrates how electrical energy can be generated from footsteps using piezoelectric sensors and monitored through an IoT platform.

The system is suitable for high-footfall areas such as:

* Railway stations
* Shopping malls
* Airports
* Schools & Colleges
* Smart cities
* Public walkways

## ⚙️ Working Principle

1. A person steps on the platform.
2. The piezoelectric sensors generate AC voltage from mechanical pressure.
3. The bridge rectifier converts AC into DC.
4. Capacitors smooth the output voltage.
5. The rechargeable battery stores the generated energy.
6. Arduino measures the voltage level.
7. IoT module sends the data to the cloud/dashboard.
8. Users can monitor power generation remotely.
